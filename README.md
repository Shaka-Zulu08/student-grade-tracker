# Student Grade Tracker

A Python menu-driven application that tracks student grades automatically.

## What it does
- Add multiple students and their marks
- Calculates average automatically
- Assigns grade letters (A, B, C, D, F)
- Saves all data to a CSV file
- Loads data back on startup so nothing is lost

## Technologies used
- Python
- CSV file handling

## How to run
1. Make sure Python is installed
2. Run `grade_tracker.py`
3. Use the menu to add, view or delete students
